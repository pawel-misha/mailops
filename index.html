Setup Guide
One-time setup · ~5 minutes · Free
1

Google Cloud Console

Go to console.cloud.google.com → Create new project.

2

Enable Gmail API

APIs & Services → Library → search "Gmail API" → Enable.

3

Create OAuth Client ID

Credentials → Create → OAuth Client ID → Web Application.

4

Add Authorized Origin

Add http://localhost:3000 as authorized JS origin (no path).

5

Add Test User

OAuth Consent Screen → Test Users → add your Gmail address.

6

Run local server

In the folder with this file run: npx serve . → open http://localhost:3000/MailOps.html

7

Anthropic API Key

console.anthropic.com → API Keys → Create key.
