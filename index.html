or add account

Set once by the app owner. Google Cloud Console → APIs & Services → Credentials.

One-time setup · ~5 minutes
1

Google Cloud Console

Go to console.cloud.google.com → Create new project.

2

Enable Gmail API

APIs & Services → Library → search "Gmail API" → Enable.

3

Create OAuth Client ID

Credentials → Create → OAuth Client ID → Web Application.

4

Add Authorized Origin

Add your site URL as authorized JS origin (no path, e.g. https://yourdomain.com).

5

Add Test User

OAuth Consent Screen → Test Users → add your Gmail address.

6

Anthropic API Key

After sign-in, open Settings (⚙) → add your key from console.anthropic.com.
